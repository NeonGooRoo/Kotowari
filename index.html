    Полностью на русском!

    • Первый полноценный гайд, написанный полностью на русском языке.

    • Содержит ссылки на источники на русском, русские словари, чаты и прочее.

    Куча программ для погружения!

    С помощью софта можно упростить потребление контента за счет моментального подсмотра слов, систематизации заучивания и прочих прелестей.

    JPDB

    Сайт для флешкарточек, датабазой контента и программами для майнинга на английском.

    Yomitan

    Подсмотр слов в браузере по наведению, а также их майнинг.

    Anki

    Приложение для эффективного заучивания слов, очень много возможностей.

    JL

    Приложение для подсмотра и майнинга слов во время просмотра аниме\игры на полный экран.

    MPV

    Плеер для майнинга и подсмотра слов.

    asbplayer

    плеер для майнинга и подсмотра слов в браузере.

    Субтитры

    программы для обработки субтитров.

    Видеоигры

    Играть в игры\вн и вытаскивать текст из них для удобного майнинга.

    ActivityWatch

    Ведет учёт открытых программ.

    OCR Инструменты

    Программы для сканирования текста с экрана.

    jidoujisho

    Всевозможная программа для Android, позволяет читать, смотреть, играть и все прочее подсматривая слова и майня.
